Building a Discord Bot using CMake/UNIX

Building with CMake

1. Toolchain

Before compiling, you will need to install cmake on your system. To be sure that cmake is installed, you can type the following command: 

$ cmake --version
cmake version 3.20.4

2. Create a CMake project

In an empty directory, create different directories and files like below: 

- your_project/
    |-- libs/
    |-- src/
        |-- main.cpp
    |-- CMakeLists.txt

In the libs/ directory, clone the sources with: git clone https://github.com/brainboxdotcc/DPP.git

3. Configure CMake

Here is an example of a CMake configuration, adapt it according to your needs:

1 # minimum CMake version required
2 cmake_minimum_required(VERSION 3.15)
3 # Project name, version and description
4 project(discord-bot VERSION 1.0 DESCRIPTION "A discord bot")
5 
6 # Add DPP as dependency
7 add_subdirectory(libs/DPP)
8 
9 # Create an executable
10 add_executable(${PROJECT_NAME}
11  src/main.cpp
12  # your others files...
13 )
14 
15 # Linking libraries
16 target_link_libraries(${PROJECT_NAME}
17  dpp
18  spdlog # if you need a logger. Don't forget to clone sources
19  # in the `libs/` directory
20 )
21 
22 # Specify includes
23 target_include_directories(${PROJECT_NAME} PRIVATE
24  libs/DPP/include
25  libs/spdlog/include # Like before, if you need spdlog
26 )
27 
28 # Set C++ version
29 set_target_properties(${PROJECT_NAME} PROPERTIES
30  CXX_STANDARD 17 # or 20 if you want something more recent
31  CXX_STANDARD_REQUIRED ON
32 )

Your project directory should look like this: 

- your_project/
    |-- libs/
        |-- DPP
        |-- spdlog # again, only if you need it
    |-- src/
        |-- main.cpp
    |-- CMakeLists.txt

Have fun!
