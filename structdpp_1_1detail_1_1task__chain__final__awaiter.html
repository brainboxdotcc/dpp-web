Detailed Description

template<typename ReturnType>
struct dpp::detail::task_chain_final_awaiter< ReturnType >

The object automatically co_await-ed at the end of a task. Ensures nested task chains are resolved, and the promise cleans up if it needs to.

Awaitable returned from task_promise's final_suspend. Resumes the parent and cleans up its handle if needed.
